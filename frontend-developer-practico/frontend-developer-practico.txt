En que consiste la maquetacion o diagrama web?
    En transformar un diseño gráfico (boceto) en una interfaz funcional en términos de programacion que entienda un navegador o dispositivo especifico.


Cual es la utilidad de un sistema de diseño?
    Facilita las tareas de diseñadores y desarrolladores en el proceso de creación. También agiliza la toma de decisión entre equipos.


Que son las variables en CSS?
    Son las propiedades personalizadas. Contienen valores especificos que se pueden reutilizar muchas veces en un documento.
    - Se establecen mediante la notacion de dos guiones:
        --nombre-variable: valor;
    - Se acceden mediante la funcion var()
        propiedad: var(--nombre-variable);
    - Normalmente se declaran dentro del selector :root para que su alcance(scope) sea global.


Cuales son las etiquetas contenedoras que nos permiten organizar imagenes de una manera semantica?
    - Figure
    - Picture


Que es el menu desplegable?
    Es un elemento de control grafico que muestra al usuario una variedad de opciones de una categoria que puede elegir para realizar una accion como una compra. Esta tiene dos estados: activa e inactiva. Cuando esta incativa, enseña un solo valor.

    
Como se aplica CSS en una orden de compra?
    Deben implementarse estilos que funcionen tanto para ventas menores como para ventas de miles de articulos. Se definiran de la siguiente manera:
        1 
            Denfinir las dimensiones del contenedor principal y centrar su contenido.
        2
            Establecer el tamaño de la fuente del titulo y su separacion de la barra de totales.
        3
            Preparar la grid de CSS Grid.
        4
            Brindar espacio a los elementos y confeccionar otra Grid.
        5
            Definir la parte visual de "shopping-cart".
        

Cual es el estilo basico de CSS para un menu?
    Se deben alinear los elementos en una barra horizontal para que ocupe el total de la pantalla.


Como se logra estilizar el menu?
    1
        Ocultar de la vista de desktop el icono del menu, dejamndolo visible solamente en la vista mobile.
    2
        Remover la decoracion de los enlaces y listas.
    3
        Centrar los elementos con flexbox, eliminar el padding y margin por defecto de las etiquetas y definir los colores y tamaños de fuentes.
    4
        Aplicar los cambios para el movil.


En el ejemplo del curso, como se centraria el carrito de compras junto al numero de articulos agregados a la orden?
    Empleando las posiciones absolute y relative.
